How to Install MySQL on your Windows 10+ Machine

Prerequisites:

  • MySQL Installer requires Microsoft .NET Framework 4.5.2 or later. If this version is not installed on the host. computer, you can download it by visiting the Microsoft Website.

Installation:

  1. Download the installer executable from the MySQL Website

  1. Run the installer by double-clicking it and allow for administrator priviliges

  2. Select the MySQL Server and Shell in the installer and click next

  1. Click Execute

  1. After Installation is complete, click next

  2. Click Finish

  3. That's it! You're Done.
